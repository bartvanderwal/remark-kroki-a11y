# Markdownlint configuration

This project recommends the [markdownlint](https://marketplace.visualstudio.com/items?itemName=DavidAnson.vscode-markdownlint) VS Code extension by David Anson for consistent Markdown style and linting.

## Usage

- The extension is recommended in `.vscode/extensions.json`.
- The configuration is in `.markdownlint.json`.
- Rule `MD034` (no bare URLs) is disabled to allow plain URLs in documentation and ADRs.
