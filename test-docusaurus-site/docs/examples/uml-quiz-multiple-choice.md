# UML Quiz (Multiple Choice)

This is the interactive multiple-choice quiz variant for the UML introduction,
written in `quiz` syntax.

```quizz
? What does a class diagram mainly show?
! Class diagrams focus on structure, not time flow.
- ( ) The order of messages over time
- (x) Classes, attributes, and relationships
- ( ) Only test results
- ( ) Only runtime errors

? What does a sequence diagram mainly show?
! Sequence diagrams emphasize message flow in time order.
- ( ) Screen layouts
- ( ) File system permissions
- (x) Interactions between actors/objects over time
- ( ) Database table definitions only

? Why is splitting the story into phases A, B, and C useful?
! Smaller focused diagrams are easier to read and maintain.
- ( ) It makes diagrams harder to maintain
- ( ) It reduces readability
- (x) It avoids one large 'God Diagram'
- ( ) It removes the need for sequence diagrams

? In the domain model, what is Basket?
! Basket is modeled as a class in the class diagram.
- ( ) An actor
- (x) A class/entity
- ( ) A sequence message
- ( ) A database
```
